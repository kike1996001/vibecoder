```

    ↓
Complexity Detection (< 100ms)
    ├─ SIMPLE → Direct generation (17s)
    └─ COMPLEX → Design questions (30-45s) + generation (60s)
    
Design Questions (if complex)
    ├─ Color palette selection (pick 1 of 5)
    ├─ Typography selection (pick 1 of 4)
    ├─ Layout selection (pick 1 of 4)
    └─ Design mockup selection (pick 1 of 3)
    ↓
Answers stored in localStorage + passed to backend
    ↓
Generation with enhanced prompt
    ├─ Status messages displayed (real-time feedback)
    ├─ Code generated by Claude
    └─ Design system info included in prompt
    
Generated code displayed in preview
    ↓
Validation triggered
    ├─ Extract preview HTML
    ├─ Validate color palette (coverage 60%+ target)
    ├─ Validate typography (correct fonts + hierarchy)
    ├─ Validate layout (pattern matching)
    └─ Validate dark mode (if selected)
    ↓
Validation report displayed (score 0-100)
    ├─ Overall score with circular indicator
    ├─ Expandable sections for each category
    ├─ Color-coded status indicators
    └─ Detailed issue descriptions
    
User can request refinements or new generation
```
